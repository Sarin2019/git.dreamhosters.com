The git merge command

git merge is used to join two lines of history.

After you fetch a remote repository, you can then merge those changes into your repository.

In this example, the remote repository named 'upstream' is merged with the local branch named 'master'.

[server]$ git merge upstream/master
Updating 1aa3afb..e3cd350
Fast-forward
 upstreamfile.txt | 1 +
 1 file changed, 1 insertion(+)
 create mode 100644 upstreamfile.txt
