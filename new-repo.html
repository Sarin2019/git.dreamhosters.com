Overview

This page explains how to create a new repository locally. This repository will only be available on your local computer.

git init

The git init command create a new git repository. This is often the first command you'll run on a new project. Run the following to create a new repository:

[server]$ git init

This creates a /.git subdirectory which contains metedata.

git init --bare

If you're working with others on this project, it's best to create it as a bare repository using the following command:

[server]$ git init --bare

For more information on why this is recommended, view this tutorial.
