Clone a repository from GitHub.com

This page details how to clone a repository from github.com.

  1. Visit github.com and locate the repository you'd like to download.
  2. On the repository page is a green button on the left titled 'Clone or download'. Click this.
  3. After you click the button, a pop-out appears below it. Copy the URL presented.
  4. On your local computer (Mac or Linux) open a terminal.
  5. Clone the repository with the following command: 
    [local]$ git clone https://github.com/repository-name.git
    • This downloads a folder containing all files in the repository.

What the clone command does

This command copies an existing repository to your local computer. The repo is fully functional so you can start working on it right away.

It also comes with a link pointed back to the original repository. You can see this by running git remote -v.
